Basics: Paragraphs and inline tags

The basic <p> tag gives this kind of text. You can add some inline formatting, where the <a> tag would give a link, the <em> tag would give some italic text, the <strong> tag would give some bold text, and the <small> tag would give some smaller stuff.

Note that paragraphs are separated by some white space when they appear on top of another.

Basics: Headings

Headings are already preformated. Use the tags <h*> as you would normally do in a HTML document. You can also add a tag <small> for a subtitle or an explanatory line.

Grid system — 12 columns to rule them all

How to use it?

You have to assign the width of a box by adding the the class span* where * can take any integer value between 1 and 12. This number corresponds to the number of columns the box will stand. By convention, we will preferably use multiples of 3 (i.e. span3, span6, span9 and span12) but when necessary, any combination whose sum is 12 is fine.

Boxes — Elementary bricks of the layout explained

In a nutshell

The general constructor for a box is similar to:

<div class="box shadow span6">
    <h3 class="box-title">Title</h3>
    <div class="row">
        <!-- some content -->
    </div>
    <div class="separating-dashed-line"></div>
    <div class="row">
        <!-- some other content -->
    </div>
    <div class="row">
        <!-- yet some other content -->
    </div>
</div>

Organization in the document

To create a box, you simply have to create a <div> with the class box. In order to match the grid system, you also have to specify the width of box thanks to one of the classes span*. And finally, add the dropped shadow thanks to the class shadow or add a solid border thanks to the class grey-border, which gives a the render of this box.

However, the use of this class will be very sparse throughout the whole website and you should (almost) always use shadow.

Inner organization

Most of the boxes should have a title. It is obtained by a <h3> tag, with the class box-title.

Each section in a box should be encapsulated in a <div> with the class row. The sections may or may not be separated by a horizontal line, thanks to the command <div class="separating-dashed-line"></div>.

Multicolum — How to display several elements in a row

Method 1: manually split the row

You can easily display a series of row elements in a simple way: Prometheus automatically manages the widths for you.

With this technique, all the items you will display in a row will automatically scale to the same width in order to fill the whole width of the box. The only parameters you have to know are the span number of the containing box (i.e. the number in the span* class) and the number of items you want to display in the row. Each object of the row should have the class item[#span]-[#items] where [#span] is the span number and [#items] is the number of items you want to display in the row.

For instance, if you want to display a row of 8 items in span6 box, each item of the row should have the class item8-6.

As another example, here is the code that was used to generate the box on the right:

<div class="box shadow span3">
    <h3 class="box-title">Title</h3>
    <div class="row">
        <img class="item3-1" src="img/ressources/flower1.jpg" alt=""/>
    </div>
    <div class="row">
        <img class="item3-4" src="img/ressources/flower2.jpg" alt=""/>
        <img class="item3-4" src="img/ressources/flower3.jpg" alt=""/>
        <img class="item3-4" src="img/ressources/flower4.jpg" alt=""/>
        <img class="item3-4" src="img/ressources/flower5.jpg" alt=""/>
    </div>
</div>

Notices

We only used images for the examples presented here but this technique can be used on any item.

The feature is currently only supported for span3, span6, span9 and span12 boxes.

Method 2: use tables

If you want to draw lines between your columns, you have to use tables. They have to be included in a container div with the class table-container, and the table must have the class split.

Although an HTML table should have the same number of columns, Prometheus allows you to tweak this rule. If you want custom row, you can stack up several tables in the container div. For instance:
